# Variable/Placeholders

| Variable     | Alias | Description          |
| ------------ | ----- | -------------------- |
| {prefix}     | {p}   | Plugin's prefix      |
| {list}       |       | List of crates       |
| {player}     |       | Player's name        |
| {crate}      |       | Crate's identifier   |
| {crate-name} |       | Crate's display name |
| {item}       |       | Item name            |
| {type}       |       | Type name            |
| {cost}       |       | Crate cost           |
| {reward}     |       | Reward String        |
